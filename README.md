***HireHub - A Job Portal Website***

This is a university group project made by 3 members. The project is called HireHub. It is a simple job portal where job seekers can apply for jobs and recruiters can post jobs.

Member 3: Halimatu Sadiya (ID: 233032012)

**Features:**

Job seekers can view jobs and apply for jobs. Recruiters can post new jobs and manage applicants. Users can register and login. Application status can be updated by recruiters.

**Technologies Used:**

Frontend: React + Tailwindcss

Backend: Node.js + Express

Database: MongoDB (Mongoose)

**How It Works:**
1. Users register as jobseeker or recruiter.
2. Jobseekers can see all jobs and apply.
3. Recruiters can create, edit, and delete jobs.
4. Recruiters can see all applicants for their jobs.
5. Application status can be updated by recruiters.
